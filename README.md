# ArkaGraph

ArkaGraph is an advanced, professional-grade keyframe interpolation and motion architecture panel for Adobe After Effects. It streamlines complex animation workflows by providing a visual, intuitive interface for curve manipulation and keyframe management.

## Features

- **Advanced Interpolation:** Multiple easing engines including Bézier, Elastic, Bounce, Wave, and Custom curves.
- **Precision Tools:** Mirror graph functionality and reference ghost curves for consistent timing.
- **Workflow Efficiency:** Seamless expression looping, smart keyframe baking, and a robust preset library.
- **Customizable UI:** Clean, theme-based interface (Carbon, Steel, Olive).

## Upcoming Updates

We are constantly improving ArkaGraph. Stay tuned for these upcoming features:
- **Custom UI Backgrounds:** Support for Image, Video, and GIF backgrounds.
- **AI-Powered Curve Generation:** Automatically match easing based on video references.
